Features

More than a basic expense tracker.

balancebook supports everyday transaction tracking, monthly review, budgets, recurring entries, separate ledgers, shared access, bank imports, and data export. This list is ordered by the features most people are likely to use first.

Fast manual entry

Record transactions with categories, tags, dates, notes, and recurring entries.

Budgets

Create category budgets and keep progress visible throughout the month.

Monthly review

Compare months, spot changes in spending, and see where money is going.

Multiple ledgers Pro

Keep personal, household, side-business, or shared finances in separate ledgers.

Categories and tags

Organize transactions with custom categories and tags, then use them later for filtering and breakdowns.

Recurring entries

Set up repeating income or expenses once and let the app handle the schedule going forward.

Bank import and review Pro

Import transactions automatically, review them before they are logged, and use rules to reduce repeat work.

Shared ledgers Pro

Share a ledger with another person and choose whether they can view it or edit it.

Cash flow and trends Pro

Use the Sankey chart, category trends, tag breakdowns, and budget history to analyze how money moves over time.

Import and export Pro

Export ledger backups as JSON, export entries as CSV or JSON, and import data when you need to restore or move it.
